# Legal & Ethical Boundaries

## Core Principle

**Public information only.** If you need a password, login, or social engineering to access it, it's off-limits.

## What You CAN Do

- Search public websites
- Read public social media posts
- Access public records and databases
- Use the Wayback Machine
- Search breach databases (read-only, checking if email was exposed)
- Read public court records
- Access company filings (SEC, Companies House, etc.)
- Analyze publicly posted images/videos

## What You CANNOT Do

- Access accounts without authorization
- Bypass paywalls or access controls
- Use stolen credentials
- Scrape at rates that constitute DoS
- Access private messages or DMs
- Impersonate someone to gain information
- Access "friends only" or restricted content
- Hack, crack, or exploit systems
- Access deleted content from private sources

## Gray Areas - Ask First

These require explicit user authorization:
- Contacting people directly for information
- Using paid OSINT services with API keys
- Accessing content that requires free registration
- Running tools that interact with live systems

## Ethical Guidelines

### Minimize Harm

- Don't expose private individuals unnecessarily
- Consider if the information could be used to harm someone
- Warn the user if findings could enable harassment

### Context Matters

Acceptable for:
- Verifying business contacts
- Due diligence on companies
- Fact-checking public claims
- Security research (own systems)
- Journalistic investigation (public interest)

Requires caution for:
- Personal investigations (relationships, etc.)
- Competitive intelligence
- Background checks (legal requirements vary)

### Report Limitations

Always note:
- What you couldn't verify
- Potential for false positives
- How information could be misused
- When findings are inconclusive

## Platform-Specific Rules

### LinkedIn
- Public profiles only
- Don't scrape at scale
- Respect robots.txt

### Twitter/X
- Public tweets only
- API rate limits apply
- Protected accounts are off-limits

### Reddit
- Public posts only
- Deleted posts may be in archives (gray area)

### GitHub
- Public repos only
- Don't clone private repos
- Respect .gitignore patterns

## When to Stop

Stop and consult the user if:
- You find potentially illegal activity
- Information could enable serious harm
- You're asked to access private content
- Legal jurisdiction is unclear
- The target appears to be a minor
