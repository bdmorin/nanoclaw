# Pattern: Summarize Profile

Synthesize findings about a person or organization into a coherent profile.

## When to Use

- After completing OSINT research on a subject
- Creating a dossier or briefing
- Summarizing background check findings

## Process

### 1. Gather Raw Data
Compile all collected information:
- Account discoveries
- Public records
- News mentions
- Social media activity
- Professional history

### 2. Verify Key Facts
Cross-reference critical claims. Note confidence level for each.

### 3. Build Timeline
Arrange events chronologically:
- Career milestones
- Relocations
- Online activity patterns
- Public appearances

### 4. Map Connections
Identify relationships:
- Professional affiliations
- Organizations
- Associates/collaborators
- Online communities

### 5. Identify Patterns
Note behavioral patterns:
- Posting frequency/times
- Topics of interest
- Communication style
- Geographic patterns

### 6. Flag Gaps
What's missing or unexplained?
- Time periods with no data
- Inconsistencies between sources
- Unverified claims

## Output Format

```markdown
# Profile: [Name/Entity]

## Summary
[2-3 sentence overview]

## Identity

| Field | Value | Confidence |
|-------|-------|------------|
| Full Name | ... | High/Medium/Low |
| Known Aliases | ... | ... |
| Location | ... | ... |
| Occupation | ... | ... |

## Online Presence

| Platform | Handle | Status | Notes |
|----------|--------|--------|-------|
| Twitter | @... | Active/Inactive | ... |
| LinkedIn | ... | ... | ... |

## Timeline

- **[Date]**: [Event]
- **[Date]**: [Event]

## Connections

- **[Name]**: [Relationship]
- **[Organization]**: [Affiliation]

## Patterns & Observations
[Notable behavioral patterns, interests, tendencies]

## Red Flags / Concerns
[Any inconsistencies, suspicious elements, or areas requiring further investigation]

## Gaps
[What couldn't be verified, missing information]

## Sources
1. [URL] - [What it provided]
2. [URL] - [What it provided]

## Methodology
[Brief description of how research was conducted]
```

## Notes

- Stick to facts; clearly label speculation
- Include confidence levels
- Don't over-interpret limited data
- Consider privacy implications
