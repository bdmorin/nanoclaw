# Contributing

## How to Help

Thank you for wanting to contribute to documenting the deportation machine. Every contribution helps shine light on what's happening.

---

## What We Need

### Priority: New Intelligence
- **Unreported deaths** in ICE/CBP custody
- **Detention facility information** — conditions, staffing, inspection reports
- **Deportation flight data** — routes, aircraft, frequency
- **Violence incidents** — raids, shootings, chemical weapons use
- **Extremist event documentation** — fascist rallies, counter-protest coverage

### Also Valuable
- **Corrections** — found an error in our data? Open an issue.
- **Additional sources** — have a source we missed for an existing dossier? Submit it.
- **Translations** — help make this available in Spanish, French, Haitian Creole, etc.
- **Data visualization** — can you make our data more accessible?

---

## How to Contribute

### Report an Issue
Open an issue if you:
- Found an error in our data
- Know of a death/incident we haven't documented
- Have additional sources for an existing case
- Have a question about our methodology

### Submit a Pull Request
1. Fork the repository
2. Create a branch (`git checkout -b add-new-intelligence`)
3. Add your contribution following our format guidelines below
4. Submit a PR with a clear description

### Email Us
Don't use git? No problem.
**oilcloth@posteo.us** — send us links, documents, corrections, or tips.

## Format Guidelines

### Dossiers
Each person documented gets a directory:
```
dossiers/YYYY-MM-DD_firstname-lastname/
├── report.md          — main report (use our template)
├── metadata.json      — structured data
└── sources.md         — all sources with URLs and access dates
```

### Source Requirements
- **Three independent sources minimum** for all major claims
- Sources must be publicly accessible (no paywalls we can't verify)
- Include URL and date accessed
- Note if a source is primary (government doc) vs secondary (news report)

### Verification Standards
- State confidence level: HIGH / MEDIUM / LOW
- Explain what's verified vs. what's still under investigation
- If you can't verify a claim to our standard, say so — we'll investigate

## What We Don't Accept

!!! danger
    - Private, classified, or stolen information
    - Content from behind authentication (must be publicly accessible)
    - Unverified claims presented as fact
    - Content that could endanger detained individuals
    - Information from law enforcement sources
    - Personally identifiable info about undocumented people (unless already public)

---

## Code of Conduct

- **Every human matters.** We document all victims regardless of citizenship, status, or history.
- **Accuracy over speed.** Get it right, not first.
- **Public sources only.** No exceptions.
- **No doxxing.** We document organizations and public figures. We don't target private individuals.
- **Respect the dead.** These were people, not data points.

---

## Review Process

All contributions are reviewed for:
1. **Source verification** — do the sources check out?
2. **Accuracy** — do the facts match across sources?
3. **Format compliance** — does it follow our structure?
4. **Safety** — could this endanger anyone?

We aim to review within 48 hours. Complex contributions may take longer.
